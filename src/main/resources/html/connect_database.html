Agregar bases de datos

Para garantizar la seguridad y los permisos adecuados, debes conectarte a tu propia base de datos. Esta herramienta no maneja datos externos a menos que configures la conexión a tus bases de datos. Responde las siguientes preguntas para configurar correctamente tu conexión. Una vez que todo esté correcto, se creará un formato JSON con los datos, que aparecerá en tu panel de integración. Asegúrate de verificar la información antes de proceder.
